# Warlord-Game
Warlord é uma recriação digital e reimaginada do jogo de tabuleiro print-and-play Mutineer (criado por Steve Smith). Nesta adaptação Warlord, o jogador deve gerenciar recursos, manipular dados e decidir estrategicamente para sobreviver a ondas de inimigos e derrotar o Senhor da Guerra.
Você pode jogar o Warlord diretamente no seu navegador através do OnlineGDB: https://onlinegdb.com/WtteQ4y9PE
